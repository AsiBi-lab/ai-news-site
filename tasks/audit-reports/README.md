## For New Agents

If you're a new agent continuing this work:

1. Read `FINAL_AUDIT_REPORT.md` first (if exists)
2. Check which reports already exist
3. Continue from the next uncompleted phase
4. Follow instructions in `../../tasks/COMPREHENSIVE_SITE_AUDIT_PROMPT.md`
